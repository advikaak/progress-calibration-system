# Progress Calibration System

## Overview
The Progress Calibration System is a Python-based project that explores how people evaluate their progress over time. Instead of relying only on visible outcomes, this project focuses on process-based actions, comparison exposure, and alignment over time.

This project was inspired by the idea that many people feel "behind" not because they are failing, but because they are using incomplete or distorted metrics to judge their progress.

## Purpose
The goal of this project is to model progress in a more realistic way by emphasizing:

- actions over outcomes
- process over external comparison
- alignment over short-term visibility

## Current Features
- calculates a process score based on actions completed
- calculates an outcome score based on visible goals completed
- adjusts progress using comparison exposure
- returns a simple progress status
